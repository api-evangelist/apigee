# Apigee GraphQL API

Apigee is Google Cloud API management platform. The API covers API proxy design and deployment, developer portals, API products, app management, API analytics, quota management, and API security policies.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Apigee REST API
**Documentation:** https://cloud.google.com/apigee/docs

- Reference: https://cloud.google.com/apigee/docs/reference/apis/apigee/rest
