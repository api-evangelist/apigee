# Apigee (apigee)

Apigee is Google Cloud's native API management platform for building, managing, and securing APIs across any use case, environment, or scale. It provides API proxies, security, rate limiting, quotas, analytics, monetization, and developer portal capabilities with full lifecycle API management — now with first-class agentic AI affordances: Model Context Protocol (MCP) support, an AI-powered Specification Boost in API Hub, and integration into Google Cloud's broader agent stack.

**URL:** [https://cloud.google.com/apigee](https://cloud.google.com/apigee)

Cataloging, organizing, and governing APIs across an organization. Powers API discovery, metadata management, dependency mapping, deployment tracking, AI-powered Specification Boost, and MCP tool registration for agentic workflows.
